Learn About Me

Programming started as a hobby for me when I was about 12 years old. I started writing code for the video game Minecraft. At first I was just learning how I could display simple text in the console and in game when the player typed a command. After learning the basics, I moved into creating server side plugins which were tools that added features to the game and could be utilized by server owners and players.


Currently I am a third year in the undergraduate computer science program at Western Washington University where I participate in a college eSports team, enjoy being apart of the snowboard class that is offered in the winter, and write code for projects that I love.
